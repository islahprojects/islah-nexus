# Islah Nexus Project Charter

## One-sentence purpose

Islah Nexus is a local-first, consent-based knowledge mirror that helps people preserve, search, and return knowledge without abandoning the vulnerable.

## Non-negotiables

1. The human remains the core.
2. AI is a compass, not the sovereign.
3. Private data is never ingested without consent.
4. User identity is never stored raw by default.
5. Every output preserves uncertainty.
6. Medical authority is never replaced by AI.
7. Walang Maiiwan: no one gets left behind.

## v0 Build Target

Build a small local command-line engine that can:

- audit an idea under the Seven Laws
- compute a NexusGPT contribution score
- store local memory in JSONL
- recall memory by keyword
- reject unsafe or exclusionary patterns

## Next Milestone

Turn the CLI into a local desktop/web interface:

- Nexus tab: ask and audit
- Notes tab: local memory
- Layers tab: laws and score breakdown
- Settings tab: privacy, export, delete
